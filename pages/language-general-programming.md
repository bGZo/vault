# 通用编程语言

在计算机软件中，通用编程语言指被设计为各种应用领域服务的编程语言。通常通用编程语言不含有为特定应用领域设计的结构。

相对而言，领域特定语言就是为某一个特定的领域或应用软件设计的电脑语言。比如说，LaTeX 就是专门为排版文献而设计的语言。
