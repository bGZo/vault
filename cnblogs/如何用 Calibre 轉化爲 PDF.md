
簡單的講，直接在操作欄選擇轉換書籍，然後選擇 PDF 即可，但是導出書籍的格式不一，最好還是有一些約定好的配置，兼顧閱讀和打印，也可以避免版權問題。

- 轉換書籍 -> 輸出格式 PDF
- 界面外觀 -> 字體 -> 最小行高 -> 180%
- 頁面設置
    * 輸出配置文件 -> 默認配置文件
    * 輸入配置文件 -> 默認配置文件
* PDF 輸出
    * 紙張大小 -> A4
    * 襯線字體 -> 思源宋體 CN
    * 非襯線字體 -> 文泉驛正黑
    * 等寬字體 -> 文泉驛等寬正黑
    * 默認字體大小 -> 16 px
    * 等寬字體大小 -> 12 px
