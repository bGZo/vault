# 群星闪耀

「你有多久没写读后感啦～」

这听起来像个蠢问题，听起来像是一种课后作业，对吗？那我换个问题：

「你有多久没写长篇的书评、游评、影评啦～」

这个听起来更怪了，我们又不是这些方面的大家，干嘛写为这写玩意（书籍、游戏、电影）写东西，看过了不就行了，那我再换个问题：

「你有没有这些东西的截图，或者表情包呢？」

如果也不曾有过，请允许我再换一个问题：

「读这些东西时，有没有一个瞬间让你动容，某个打破荧幕，照见自己的时刻？」

如果你也曾感受过这样的一瞬之光，那你一定懂，珍贵的地方就在这短短的一瞬，如果你不伸手够它，他就会慢慢消失，知道你再也回忆不起他是什么。

我当然做过很多书摘，对书籍抽丝剥茧，输出摘要，但都无甚用处，日后再看内心也难起波澜，甚至有时候想想要不删了更好，加上多年的积累，数量已经庞大到难以整理的程度，这其中一定有很多没有必要的东西，但也一定有对等重要的东西值得记录。

照见自己内心的某个角落，看到某个角度的自己，并记录下来，这就是这里存在的理由。

但如果你从来都不曾有过这样的体验，我确实可以承认你是一个资本主义社会下完美的消费者，也许你可以看看这里的东西，可能也会对你有启发。

总而言之，谢谢你看到这里，希望这里的东西对你有用。
