# Nuphy Air75 黛粉轴

## 动机

1. 价格便宜，蹲了很久，价格从 679（京东） -> 539（抖音）
2. 工位和宿舍各需要一把键盘，来回背有点不现实；

### 爽点

1. 静音，黛粉声音几乎没有；
2. 相比于 [[1268-keychron-z7max|Keychron Z7Max]] 的手感要轻松很多；
3. 键帽不容易翻车了，稳定性好了一些；
4. 2.4G 接收器终于可以收纳到机器内部了；

### 不爽的点

1. 更重了；
2. 没有塑料防尘罩；
	1. 可能是自定义旋钮不太好做壳；
	2. 可能是推销自家的保护皮套（100+）；
3. 轴体与前两代完全不一样，无法通用；
