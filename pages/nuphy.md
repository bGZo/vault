## 快捷键

- 电量显示：`FN + \`
- Air96
    - 侧灯：`FN + ? + ∧ / ∨ / < / >`
    - F 键功能切换：` FN + X + F`
    - 睡眠模式：`FN + X + C/V`
        - 闪绿灯 => 开启
        - 闪红灯 => 关闭
        - 下适用
- Air75 V2
    - 侧灯：`FN + M + ∧ / ∨ / < / >`
    - 睡眠模式：`FN + ]`
