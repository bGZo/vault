# Keychron Z7Max

## 挑挑拣拣

先来说说为什么要入手这款键盘吧：

1. 价格
2. 电池

价格是绕不开的因素，因为我现在服役的是 Nuphy AirV2 芦荟 + 越菊轴。这把键盘总是放在公司，家里面还有无数的电子设备，在关键时候总是少一把键盘，所以总想着能入手一把备用键盘。

然后我路径依赖地看了最近发布的 Nuphy AirV3，粉黛轴我觉得自己应该适合，触发力度应该比芦荟要高一点，应该适合我这种肌无力的群体。但是价格全新的仍然要 670+，比预售的还要贵 50。

所以只能看看市面上的平替了：

- Nuphy Kick75
- Nuphy Node75
- [洛斐小顺 Flow2](https://www.lofree.co/products/flow-2-68-low-profile-mechanical-keyboard) / 青春版
- [Keychore K3 Max](https://www.keychron.com/products/keychron-k3-max-qmk-via-wireless-custom-mechanical-keyboard)

一个个说，Kick75 颜值更高（半透明 + 像素）、客制化更高（该爱周切换） ，但是我发现他没有 Insert 键，右上角的那个旋钮更是无法更换成按键，对我来说完全是个鸡肋。

再来，Node75 是一个价格更加亲民的选择，但是右上角的触控条和洞洞的设计我同样不喜欢，触控条误触是一方面，容易坏是另一方面，坏了怎么办，没有办法修（无法完美拆开，拆开必出伤 [^hurt-when-fix]）；洞洞看着就像是个藏污纳垢的地方，所以不太感冒。

[^hurt-when-fix]: https://www.bilibili.com/video/BV1pg1iBHEu2, Node75 前高同样多 5mm， http://xhslink.com/o/9JRoBgEB3Wc, 导致最终打字感受也会大一点。

接着看洛斐，洛斐是我身边同事用的比较多的牌子，但对我来说比较陌生，全铝的 FLow 看着很好看，但是右侧的触控条同样不感冒，青春版上面的旋钮塑料感满满，依然不感冒，我有点习惯放在 Mac 笔记本上面的一致性了，所以同样有点难以接受，直接不看了。

看了看去，最后看到了渴创，我记的这个牌子最早是从 AK 口里面知道的，但一直没有接触，后面搜了一下，K3Max 固定轴价格真是太便宜了，300 块之内就能拿下全新的，热插拔也只会一点点，但仍然有问题：轴体就三个，电池就 1500ma，半个月就需要充电一次。后来我也直接放弃了。

后面继续看看最近的新品，刷到了 Z7max，虽然轴体也只有定制凯华白轴这一种 [^kaihua-bai-zhou]，但是电池来到了 4000ma，跟 Nuphy 差不多，价格上也便宜不少，最终能做到 400。

[^kaihua-bai-zhou]: 后面跟客服对线才了解到 Z7Max 的轴体是厂商特地定制的，失眠上有售的凯华白轴压力是 50+-10g，这一款的产品标注页是 40+-10g。

这个时候我已经很累了，所以直接下手了，充其量就是 K3Max 红轴的加强版，况且也只是一把备用键盘而已。

## 意外之喜

到手伤手体验了一把之后，感受了一些惊喜之后，才发觉出之前 Nuphy 的一些问题，先来看看那些点让我比较惊艳：

第一点，全铝机身/铝坨坨做工；全铝机身没得说，整体的质感上来了，比 Nuphy 的半铝半塑料摸起来好很多 [^nuphy-air-su-liao]；

[^nuphy-air-su-liao]: 不确定是不是问题，如果全铝放在 Mac 上面，会不会有刮花的问题？

第二点：2.4G 接收器收纳仓（V3 已经跟进）；Nuphy Air75V1、V2 都没有 2.4G 接收器的收纳仓，只能自己放包里，所以这个直接收纳到背部的设计一定是好设计；

第三点：网页驱动（V3 已经跟进）；KeyChore 直接就可以在线升级驱动，改建，等等操作，我后来也尝试升级下手里面这把 V2 的固件，但乱八七糟的指南，加上网上的教程极少，所以最终放弃，反正还能正常用，也不那么在意了。

第四点：Mac 适配，有一个小细节，我用有线连接 Mac 的时候，Mac 立马就识别出来这把键盘是 Keychron 的，而 Nuphy 就什么提示都没有，我不清楚其中的实现原理，可能 Keychron 跟 Apple 有什么交易吧。

第五点：轴体稳定性篇；代工厂不同，Nuphy 是佳达隆代工，Keychron 是凯华代工，本质上就是两款轴的做工 PK，这一点凯华表现更稳，就算按的角度刁钻，也不会像 Nuphy 那样把键帽直接飞出去。

写了这么多，我自己也发现好像自己对 AirV3 的质感和价格不满意，我是全面向往的 😂 喔对了，Nuphy 还有溢价的问题，尤其是最近几年 Nuphy 的迭代速度更快了，二手市场的价格跳水不一，比如我这款 Air75V2 二手价格已经跌了快一半了（350 左右），我是不愿意当「首发用户」。

## 不足之处

好处说了很多，坏处也不少：

第一点：没有脚撑，虽然设计上已经给了用户足够的坡度，但是没有脚撑就是没有脚撑，在很多时候我的键盘下面是要垫一根木棍的，这样打起字来才舒服；

第二点：轴体不习惯，我是从 Nuphy 芦荟 + 越菊换过来的，他们的触发压力大概差不多：

- 芦荟轴：37gf+-15gf
- 越菊轴：45gf+-15gf
- 白雨定制：40gf+-10gf

但是实际上我拿到手的打字感觉上，z7max 的压力是要比 air75 更重的，三种轴的感觉完全不一样，总体上用 z7max 打字更加费力，不如 Nuphy 用着习惯 [^zhou-ti]。

[^zhou-ti]: 我不知道是不是我的错觉， 芦荟轴有点用了就回不去的感觉，现在我连红轴都感觉压力有点大， https://v2ex.com/t/1013669, https://v2ex.com/t/1013669

第三点：花里胡哨的键帽，我一开始以为自己对键帽不是很在意，因为我平时敲代码就是盲打，但盲打不是完全不看键帽，偶尔看到那个键帽真的有点视觉污染，完全找不到北，这一点真的不如简约的键盘来的方便（那个橘色的符号瞅着太费眼睛了，得稍微努力一点才能识别）。

第四点：无线死机 [^die-suddenly]，我不确定这个问题未来时候还会再次复现，我平时用 2.4G 模式，在敲着敲着的时候，键盘就没有反应了，那种状态非常微妙，大小写不亮，任何功能键失效，插电电源灯能指示充电，然后无线模式也连不上，唯一解就是在不插电的情况下完全关机，静置几秒重新开启，这样才能好，我实在搞不清楚这是为什么，尤其是在我打字的时候突然来这么一下非常糟心。

[^die-suddenly]: 我怀疑是驱动问题，我昨天升级了最新驱动，还需要在观察一下。

第五点：没有办法完美的放在 Mac 上，虽然可以放在 Mac 上，但这一点不如 air75 来的顺心。[^laptop-above]

[^laptop-above]: 这个使用场景还需要考量，因为 Windows 笔记本电脑上的发热比 Mac 大，热量会直接导到电池上，电池的寿命可能因此大打折扣。

第六点，不知道算不算问题的问题，Z 系列不在海外上市，也就是国内特供，这个不确定官方是什么考虑，但未来的支持一定是会弱一些的。

第七点，没有防尘罩，至少 Nuphy 还送了一个塑料的壳子，日常不用的时候也能罩在上面，不落灰， 而 z7max 没有，甚至 air75V3 之后也没了，我不理解，就算后期自己加这个东西也很难买，我找了半天，只能再买个通用的亚克力防尘罩，适配度也不高。

## 牢骚两句

最近为了买个备用键盘真是看了太多软文了，现在我几乎分不清哪些主播说的是实话，哪些是打广告，哪些是拉踩的了，这方面太乱了。

二手价格市场是另一方面，如果你手头拮据，又想买把自己用，就会发现闲鱼上面有大量的新品卖的很便宜，问有没有订单售后就支支吾吾，有些卖家直白，直接说是厂商送测的，有部分就不说，还跟官方卖正常的价格，这方面水也比较深，一不小心就会买到无售后的版本，需要仔细甄别。

因为矮轴没有统一的方案，所以基本上都是各玩各的，只能在厂商的几种有限的方案里面折腾，比较难受，真希望来个秦始皇能统一这种布局。

也许明年我又会去买 V3 的二手了，也许不会，但那都是明年的事情了，暂时就这样吧。
