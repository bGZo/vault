# 如何同时和 AI 聊天

市面上有非常多 ai chat 的产品，大部分提供网页端的使用渠道，每次如果想看看别的 ai 的回答就需要重复输入一遍上下文，非常烦，非常需要一个产品可以只输入一次，然后多个产品集体响应？

目前用的最舒服、最符合逻辑的是 https://addons.mozilla.org/zh-CN/firefox/addon/multi-search-ai ，但依然有 BUG，比如 Chatgpt 可以输入，但不会自动发送，Deepseek 无法自动输入，需要重新输入一遍，Gemini 和 Copilot 用着完美。

还有一些用过比较反直觉，反人类的，比如开一堆标签页，然后再问一句，关闭原来的标签页，重来一遍，有：

- https://chromewebstore.google.com/detail/multi-ai-chat/ajjjgkcggdkpaomfpghendbngmkfighe
- https://addons.mozilla.org/zh-CN/firefox/addon/multillm

当然如果家境富裕，可以使用付费产品，或 API Token，这是最简单，最直接的，比如：

- https://multiple.chat/subscription
- 

## Firefox 使用场景

```shell
我是一位 LLM 新手，请给我一份 LLM 学习指南，我的目的是在空闲时间学习，好让自己，的LLM技能在未来找工作（编程）时有优势

我现在只知道 Prompt工程对我可能有用，尽量让指南详细，可达成，可量化，可考量。
```
