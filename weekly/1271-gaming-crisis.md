# 游戏危机

最近不是把 SD 都卖掉了吗？可我还是想要打游戏，所以不甘心，下意识就开始打起了其他掌机的主意。

我看 Ally X，Ally Xbox X，Switch OLED 这些，看了很久，但都没有下手。

首先，我卖掉 [[1269-giving-up-the-offical-version-on-steam-use-cheat-version-again|SD]] 是因为它没有办法很好的玩盗版游戏，很多游戏需要依赖 Proton 兼容层，如果只装这个就算了，但是你还是得装个 Steam，可以说，Linux 下一半都游戏体验都是 Steam 给的，如果不在 SD 上启动 Steam，你连键盘输入法都打不开（steam+x），Steam 为 Arch 做的一切，都离不开 Steam 这个平台，这意味着什么？

你每次都需要启动 Steam，需要联网，需要等待，甚至跟 Windows 自动更新一样，需要等待 Steam 漫长的进度条， 如果网络不好，系统随时有挂掉卡 Logo 的风险，还需要手动回退上个进本的系统，然后再进行操作。另一方面，你玩个 Steam 平台没有上架的游戏，也麻烦的要死，

想办法在 Linux 里下载游戏，很多网盘没有 Linux 客服端，下载完了解压还是个麻烦事，你得保证解压后的游戏不会出现奇怪的乱码（实际上大部分会），然后打开 Steam 客户端进行导入，成功后还不一定能启动，需要切换不同的兼容层一次次尝试，而且就算成功启动了，也并非万事大吉，有可能字体发虚、方块，各种各样头疼的问题。所以我想要一个基于 Windows 的掌机。

但 Windows 掌机就好了吗？不是，Windows 掌机是另一种折磨人的方式：首先，系统占用，如果一个系统开机就要占用一半以上的系统资源，甚至还会跟游戏抢夺它的话，我是无法接受的，但这就是目前 Win 掌机的现状。

无止尽的后台与服务，让这台掌机跟笔记本、台式机顶着相同压力跑游戏，散热还远远不如笔记本和台式机，这也是我最担心的问题，不是游戏帧数上不去，我可以接受 30 帧的游戏体验，但你掌机不能玩几年就坏了吧？ Ally 初代烧 TF 卡的问题简直是一场灾难，2232 尺寸的固态发热量又大，那款掌机很难保值。两年前还是 4000～5000 的价格，现在看了眼，已经到了 2300～2500 的程度。

Ally Xbox X 这款掌机光看配置和硬件，其实是一个完美答案，但是问题就是他的价格已经直逼笔记本甚至台式机，但后者能给的游戏体验，是掌机能给的吗？Xbox 定制的前端页面是好，但它和 Windows 的切换也不是无损的，你可以从 Xbox 切到 Windows ，但是如果从 Windows 回到 Xbox 页面，你得重启机器，这点不如 SD。

目前最佳的机器其实就是去年（2024 年中）发布的 AllyX，是初代的半代升级，他改进了存储卡的位置，避免了烧卡的位置，也调整了硬盘的尺寸，用上了 2280 长度的固态，电池也翻了倍，价格比 Xbox 款要更亲民，市场价 24+1T 大概要 4000，虽然还是很贵，但已经是挑不出毛病的机器了。

可是，Z1Extreme 和 Z2Exterme 都是什么 CPU？甚至都不是 Zen4 架构，这意味着什么？AMD 已经证明了这两袋架构的失败性，换句话说，如果下一代 U 一发布，这两个 U 的价格就要腰斩。所以最后我哪个都没有买。

Windows 掌机很烂，是的，费电、发热、性能释放甚至还不如 Proton 的兼容层，每个游戏同样需要不同程度的调优配置，如果你有这方面的强迫症，这一定是最麻烦的事。

统一的硬件、开箱即玩的配置、散热 OK，不会特别热的机器，我把目光移向了 Switch，虽然二代已经发布了，但是目前还没有破解的消息，所以我目前只会考虑一代的机器。

这方面的水其实更深了，BAN 机，双系统升级，看看现在市场价，全新的机器百亿补贴完 1450，加上 100 的破解费用，全新破解的机器差不多就 1550，1600 的样子，二手市场价格同样虚高，理论上二手市场几年前的机器应该来到 1200～1400 左右，但是市场价不是这样的，闲鱼的价格普遍在 1600～2000 这个区间，想要收一个二手太难了。

大多数时候，我只是想要好好地玩个游戏，不像折腾这些有的没的，Switch 当然只是个理论上的最优解，但他好像更像是个围城，我还没有太多的涉猎，也许未来会试一试。
