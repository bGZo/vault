最近不是把 SD 都賣掉了嗎？可我還是想要打遊戲，所以不甘心，下意識就開始打起了其他掌機的主意。

我看 Ally X，Ally Xbox X，Switch OLED 這些，看了很久，但都沒有下手。

首先，我賣掉 SD 是因爲它沒有辦法很好的玩盜版遊戲，很多遊戲需要依賴 Proton 兼容層，如果只裝這個就算了，但是你還是得裝個 Steam，可以說，Linux 下一半都遊戲體驗都是 Steam 給的，如果不在 SD 上啓動 Steam，你連鍵盤輸入法都打不開（steam+x），Steam 爲 Arch 做的一切，都離不開 Steam 這個平臺，這意味着什麼？

你每次都需要啓動 Steam，需要聯網，需要等待，甚至跟 Windows 自動更新一樣，需要等待 Steam 漫長的進度條， 如果網絡不好，系統隨時有掛掉卡 Logo 的風險，還需要手動回退上個進本的系統，然後再進行操作。另一方面，你玩個 Steam 平臺沒有上架的遊戲，也麻煩的要死，

想辦法在 Linux 裏下載遊戲，很多網盤沒有 Linux 客服端，下載完了解壓還是個麻煩事，你得保證解壓後的遊戲不會出現奇怪的亂碼（實際上大部分會），然後打開 Steam 客戶端進行導入，成功後還不一定能啓動，需要切換不同的兼容層一次次嘗試，而且就算成功啓動了，也並非萬事大吉，有可能字體發虛、方塊，各種各樣頭疼的問題。所以我想要一個基於 Windows 的掌機。

但 Windows 掌機就好了嗎？不是，Windows 掌機是另一種折磨人的方式：首先，系統佔用，如果一個系統開機就要佔用一半以上的系統資源，甚至還會跟遊戲搶奪它的話，我是無法接受的，但這就是目前 Win 掌機的現狀。

無止盡的後臺與服務，讓這臺掌機跟筆記本、臺式機頂着相同壓力跑遊戲，散熱還遠遠不如筆記本和臺式機，這也是我最擔心的問題，不是遊戲幀數上不去，我可以接受 30 幀的遊戲體驗，但你掌機不能玩幾年就壞了吧？ Ally 初代燒 TF 卡的問題簡直是一場災難，2232 尺寸的固態發熱量又大，那款掌機很難保值。兩年前還是 4000～5000 的價格，現在看了眼，已經到了 2300～2500 的程度。

Ally Xbox X 這款掌機光看配置和硬件，其實是一個完美答案，但是問題就是他的價格已經直逼筆記本甚至臺式機，但後者能給的遊戲體驗，是掌機能給的嗎？Xbox 定製的前端頁面是好，但它和 Windows 的切換也不是無損的，你可以從 Xbox 切到 Windows ，但是如果從 Windows 回到 Xbox 頁面，你得重啓機器，這點不如 SD。

目前最佳的機器其實就是去年（2024 年中）發佈的 AllyX，是初代的半代升級，他改進了存儲卡的位置，避免了燒卡的位置，也調整了硬盤的尺寸，用上了 2280 長度的固態，電池也翻了倍，價格比 Xbox 款要更親民，市場價 24+1T 大概要 4000，雖然還是很貴，但已經是挑不出毛病的機器了。

可是，Z1Extreme 和 Z2Exterme 都是什麼 CPU？甚至都不是 Zen4 架構，這意味着什麼？AMD 已經證明了這兩袋架構的失敗性，換句話說，如果下一代 U 一發布，這兩個 U 的價格就要腰斬。所以最後我哪個都沒有買。

Windows 掌機很爛，是的，費電、發熱、性能釋放甚至還不如 Proton 的兼容層，每個遊戲同樣需要不同程度的調優配置，如果你有這方面的強迫症，這一定是最麻煩的事。

統一的硬件、開箱即玩的配置、散熱 OK，不會特別熱的機器，我把目光移向了 Switch，雖然二代已經發布了，但是目前還沒有破解的消息，所以我目前只會考慮一代的機器。

這方面的水其實更深了，BAN 機，雙系統升級，看看現在市場價，全新的機器百億補貼完 1450，加上 100 的破解費用，全新破解的機器差不多就 1550，1600 的樣子，二手市場價格同樣虛高，理論上二手市場幾年前的機器應該來到 1200～1400 左右，但是市場價不是這樣的，閒魚的價格普遍在 1600～2000 這個區間，想要收一個二手太難了。

大多數時候，我只是想要好好地玩個遊戲，不像折騰這些有的沒的，Switch 當然只是個理論上的最優解，但他好像更像是個圍城，我還沒有太多的涉獵，也許未來會試一試。
