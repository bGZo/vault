# 命令行

## 默认颜色

| color    | decribe                                                 |
| -------- | ------------------------------------------------------- |
| **白色**   | 普通文件                                                    |
| **蓝色**   | 目录                                                      |
| **绿色**   | 可执行文件                                                   |
| **红色**   | 压缩文件                                                    |
| **浅蓝色**  | 链接文件                                                    |
| **红色闪烁** | 链接的文件有问题                                                |
| **黄色**   | 设备文件                                                    |
| **灰色**   | 其他文件                                                    |
| **绿色**   | 是有问题的，代表权限中有其它组权限拥有写入权限，系统默认这是一个高风险目录。将权限改到 775 以下就可以解决 |

## Angle brackets

`>` -> 输出重定向

`>>` -> 追加
