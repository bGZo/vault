# 如何串流云音乐

听起来很蠢，因为网易云、AM、Spotify 不就是做这个的吗？有几点原因：

1. 服务并不是完全免费的，无数的广告和订阅非常烦人；
2. 公司并不一点稳定，如果有一天全部倒闭了，那么你的数据全都丢了；
3. 不可避免或多或少的审查；

基于上述理由，想要纯粹的听歌，你总是会走到这一步的。

拆解问题，你需要一个地方存音乐，所以你需要一个网盘，但有容量的限制；如果你有跳板机当然可以内网穿透，没有容量的限制。常用的网盘就只有

- Onedrive
- GDrive

然后，你需要一个像样的前端， [[pure-music-PureMusic|PureMusic]] 是一个，但是它付费，并不免费。其他的服务我还没有找到像样的解决方案。
